# 更新日志

本项目的所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-01-15

### 新增
- 🎉 首次发布 Ai淇橦学排版预览插件
- ✨ 实现左右分栏界面设计
  - 左侧：基础设置 + 管理导航
  - 右侧：动态内容展示区
- ✨ 基础设置功能
  - 默认模板选择
  - 默认字号设置
- ✨ 效果预览功能
  - 实时预览当前设置的模板效果
  - 支持滚动查看完整预览内容
- ✨ 模板管理功能
  - 导入/导出模板
  - 新建自定义模板
  - 模板显示选项（隐藏/显示）
  - 模板预览、编辑、删除
- ✨ 字体管理功能
  - 添加自定义字体
  - 编辑字体配置
  - 删除字体
- ✨ 背景管理功能
  - 新建自定义背景
  - 背景显示选项（隐藏/显示）
  - 编辑背景配置
  - 删除背景
- 🎨 现代化 UI 设计
  - 渐变色区块标题
  - 平滑动画过渡
  - 响应式布局适配
  - 优雅的导航按钮设计

### 技术细节
- 使用 TypeScript 开发
- 基于 esbuild 构建
- 遵循 Obsidian 插件开发规范
- MIT 开源许可证

### 文档
- 完整的 README.md（中英文）
- 详细的贡献指南
- GitHub Issue 和 PR 模板
- 自动化发布工作流
